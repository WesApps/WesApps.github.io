ABOUT

Wes App provides useful, up-to-date information about Wesleyan University and current campus happenings. All of the information featured by Wes App is provided by WesAPI, a student run service for programmatic collection and access to Wesleyan data. Wes App is open source and was created by Aaron Plave 15'. This project is still under heavy development so styles, features, and layout may change from time to time. If you have any questions, bug reports, or would like to contribute to this project in any way, please send an email to aplave@wesleyan.edu. Check out the source code on Github!
